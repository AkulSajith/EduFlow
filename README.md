# EduFlow 📚

EduFlow is a role-based syllabus management and academic tracking system designed to streamline course planning, progress monitoring, and academic workflow management.

The project provides separate dashboards and workflows for administrators, teachers, and students through an interactive frontend built with React and TypeScript.

<hr>

## Features

- Role-based dashboard system
- Syllabus and course tracking
- Academic workflow management
- Progress monitoring interface
- Interactive frontend navigation
- Structured component-based architecture

<hr>

## Tech Stack

### Frontend
- React.js
- TypeScript
- Vite
- CSS

### Concepts & Architecture
- Role-Based Access Control
- SDLC Principles
- Component-Based Design
- Context API

## How to Run the Project

### 1. Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/eduflow-app.git
```

<hr>

### 2. Navigate to Project Folder

```bash
cd eduflow-app
```

<hr>

### 3. Install Dependencies

```bash
npm install
```

<hr>

### 4. Run the Project

```bash
npm run dev
```

Application runs on:

```bash
http://localhost:5173
```
